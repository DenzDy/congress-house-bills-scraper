# congress-house-bills-scraper
Automated web scraper for Congress and House Bill Resolutions from the [Philippine Congress website's](https://congress.gov.ph/) Legislative Documents section.

## Tech Stack
- **Programming Language**: Python
- **Libraries:**
    1. [Playwright Python](https://playwright.dev/python/docs/library)
    2. [Camoufox](https://github.com/daijro/camoufox)
    3. [python-dotenv](https://github.com/theskumar/python-dotenv)
- **Virtual Environment**: [uv](https://docs.astral.sh/uv/)
## Installation/Setup
1. **(Optional)** install [uv](https://docs.astral.sh/uv/) and create a virtual environment
    i. Install **uv** from the instructions in the [uv website](https://docs.astral.sh/uv/)

    ii. Create a new virtual environment via. the command below:
    ```
    uv venv
    ```

    iii. Activate the virtual environment
    ```
    source .venv/bin/activate
    ```

2. Install project dependencies using the following command:
    ```
    pip install -r requirements.txt
    ```
